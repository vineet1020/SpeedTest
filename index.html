This is a game to test your speed and reflex.

RULES:
  • The game starts with a matrix having numbers from 1 to 25 randomly arranged.
  • The aim is to select all numbers from 1 to 50 in strictly ascending order,as quickly as possible.
  • From 1 to 25,on selection of each number a new random number will be generated.From 26 onwards,the cell would become blank.
